Kdo jsem a...

kde to všechno začalo?

S kódováním jsem přišla do kontaktu již na základní škole. A už v té době mě nadchlo, že díky HTML a CSS dokáže člověk vytvořit funkční a esteticky přitažlivý web. Ale bohužel jsem se vydala jiným směrem.

A poslední dva roky se snažím na sobě intenzivně pracovat a plnit si své sny. A tvorba webu v HTML a CSS je jedním z nich. Jsem si vědoma, že mám před sebou dlouhou cestu, ale cesta je cíl a jen trénink dělá z člověka mistra.

Proto každý projekt je pro mě osobní záležitostí a velkou výzvou, jelikož weby tvořím od nuly - tedy žádné šablony ani redakční systémy. Kromě napsání unikátního kódu vám pomohu i s jeho umístěním na web - registrace domény vč. vhodného hostingu.

Tak pojďme společnými silami rozjet váš sen...
